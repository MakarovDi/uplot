Unified API for Python plotting libraries.

## Supported

- matplotlib
- plotly

## Installation

```bash
pip install git+https://github.com/makarovdi/uplot.git
```

## Dependencies

- Python >= `3.10` 
- numpy

### Optional
- matplotlib > `3.0`
- plotly > `5.0`
